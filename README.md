# Mall Customer Segmentation Analysis

## Project Overview

This project uses Machine Learning clustering techniques to segment mall customers based on their purchasing behavior and demographic information.

Customer segmentation helps businesses understand different customer groups and create targeted marketing strategies.

## Dataset

The dataset contains information about mall customers, including:

- Customer ID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1-100)

## Objectives

- Explore customer demographics
- Perform data preprocessing
- Identify customer segments
- Apply clustering algorithms
- Visualize customer groups
- Generate business insights

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Machine Learning Algorithms

### K-Means Clustering
Used to identify distinct customer segments.

### DBSCAN
Density-based clustering for customer grouping.

## Key Insights

- Customers were grouped based on spending behavior and income.
- High-income, high-spending customers were identified as premium customers.
- Low-income, low-spending customers formed a separate segment.
- Clustering helps businesses design personalized marketing campaigns.

## Project Workflow
